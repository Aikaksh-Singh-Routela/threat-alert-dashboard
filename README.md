# Threat Alert Dashboard

A full-stack threat alert management system with React frontend and FastAPI backend.

## Features

- User registration and JWT authentication
- Real-time threat alert dashboard
- Statistics cards (total, suspicious, normal, percentage)
- Create, view, and delete alerts
- Responsive design

## Tech Stack

- **Frontend**: React, Axios
- **Backend**: FastAPI (Python)
- **Database**: SQLite
- **Authentication**: JWT

## Quick Start

### Backend Setup

```bash
cd threat_alert_api
pip install -r requirements.txt
